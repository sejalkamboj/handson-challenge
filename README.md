# Round-1 Task 

## Features
- Floating animated cards using requestAnimationFrame
- Reusable FloatingCard component with color, rotation, icon, label props
- Portal card variant with glassmorphism (backdrop-filter: blur)
- Hover lift effect on all cards
- Dark mode toggle
- Responsive layout (cards hidden on mobile)
- Entrance fade-up animation

## Tech Stack
- Next.js 14 (App Router)
- React hooks (useState, useEffect, useRef)
- Plain CSS-in-JS (no Tailwind needed for animations)
